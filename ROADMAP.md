# Roadmap & Strategic Evolutions

1. **Logic Surface**
   - Expose state machines as JSON.
   - Allow non-destructive behavior patches.

2. **Telemetry Feedback Loop**
   - Instrument wrappers.
   - Analyze via telemetry service and agents.

3. **Procedural Animation Generator (RiveSynth)**
   - Agents output motion specs.
   - Generate draft assets for Rive import.

4. **Collaboration & Versioning**
   - Treat motion artifacts like code (branches, PRs).
   - Agents submit motion PRs.

5. **Scene Graph Composition**
   - Compose multiple Rive components into orchestrated scenes via specs.
